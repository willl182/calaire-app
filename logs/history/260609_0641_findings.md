# Technical Finding / Milestone

**Context**: SGC Dashboard Layout Alignment

## The Discovery

The SGC summary page can share the exact same shell as the other admin dashboard views without losing its own internal content structure. The stable pattern is:

- outer page wrapper: `min-h-screen px-6 py-8`
- inner container: `mx-auto flex max-w-7xl flex-col gap-6`
- shared identity block: `header-bar`
- SGC-specific content below that wrapper

The width mismatch that appeared in `/dashboard/sgc` was not a routing issue. It was caused by the SGC composition allowing the internal board section to expand the visual footprint. Constraining the SGC children with `min-w-0` and `w-full` restored the same measured width as `Participantes`.
