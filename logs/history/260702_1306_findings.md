# Technical Finding / Milestone

**Context**: Drive documental SGC - fase 7

## The Discovery

El cierre documental debe combinar dos fuentes de verdad: el checklist SGC existente y el expediente Drive por ronda. La regla implementada deja el editable como bloqueo principal: los recursos requeridos deben tener enlace editable o una justificacion `no_aplica`, y los formatos criticos deben estar `diligenciado`. El definitivo se mantiene como advertencia porque el plan lo define como recomendado, no bloqueante.

El helper `collectDriveCierreCalidad` centraliza esa evaluacion para que el panel y las mutaciones de transicion usen la misma lista de bloqueantes y advertencias.
