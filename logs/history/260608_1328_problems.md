# Problem Record (Troubleshooting)

**Severity**: Medium

## The Problem

The new Playwright SGC spec initially failed on `/dashboard/rondas/kd7b0emdk7cmzp1vn34f2bfv7986bb77/sgc` with:

```txt
This page couldn't load
TypeError: Cannot read properties of undefined (reading 'length')
```

The stack pointed to `panel.comentarios.length` in `app/(protected)/dashboard/rondas/[id]/sgc/page.tsx`.

## The Solution / Workaround

`lib/sgc/index.ts` now normalizes optional/new Convex payload fields:

```ts
comentarios: raw.comentarios ?? []
notificaciones: raw.notificaciones ?? []
resultadosPtApp: raw.resultadosPtApp ?? []
```

This keeps the UI compatible while Convex deployments/functions are catching up or when existing environments return the older panel shape.

## Learnings to Preserve

- For Convex-backed panel payload expansions, make wrappers tolerant to missing newly-added fields.
- Playwright must run outside the sandbox in this environment because Chromium fails with `sandbox_host_linux.cc: Operation not permitted`.
- Use `PLAYWRIGHT_BASE_URL=http://localhost:3000` when relying on the existing authenticated WorkOS session.
