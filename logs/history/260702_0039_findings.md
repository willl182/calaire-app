# Technical Finding / Milestone

**Context**: Drive documental SGC Fase 2

## The Discovery

La UI Drive puede integrarse como Server Component dentro de la pagina SGC existente usando wrappers de `lib/sgc` y Server Actions page-locales. No hace falta una ruta `/drive` paralela ni importar Convex desde la UI.

Al registrar una version definitiva, `upsertDriveRecurso` debe preservar el estado si el editable no cambia. Si no, un recurso `diligenciado` puede volver a `creado`, lo cual contradice el workflow donde el definitivo no cambia el estado del editable.

- `app/(protected)/dashboard/rondas/[id]/sgc/DriveDocumentalSgc.tsx`
