# Technical Finding / Milestone

**Context**: Migración Supabase → Convex — Fase 3: Funciones Convex

## The Discovery

Se crearon los tres archivos de funciones Convex que reemplazan todas las queries SQL de Supabase. TypeScript compila sin errores con `npx tsc --noEmit`.

### Patrones establecidos

**Queries encadenadas en lugar de JOINs:**
```ts
// En lugar de SELECT ... JOIN ronda_contaminantes ON ...
const contaminantes = await ctx.db
  .query('rondaContaminantes')
  .withIndex('by_ronda', (q) => q.eq('rondaId', ronda._id))
  .collect()
```

**Upsert con índice compuesto:**
```ts
// Equivale a ON CONFLICT (ronda_participante_id, pt_item_id, sample_group_id) DO UPDATE
const existing = await ctx.db
  .query('enviosPt')
  .withIndex('by_participante_item_group', (q) =>
    q.eq('rondaParticipanteId', rondaParticipanteId)
     .eq('ptItemId', ptItemId)
     .eq('sampleGroupId', sampleGroupId)
  )
  .unique()
if (existing) {
  await ctx.db.patch(existing._id, { ...campos, updatedAt: now })
} else {
  await ctx.db.insert('enviosPt', { ...campos })
}
```

**Cascada manual (ON DELETE CASCADE):**
```ts
// En deletePTItem — antes de borrar el item, borrar sus envios_pt
const envios = await ctx.db
  .query('enviosPt')
  .withIndex('by_pt_item', (q) => q.eq('ptItemId', itemId))
  .collect()
await Promise.all(envios.map((e) => ctx.db.delete(e._id)))
await ctx.db.delete(itemId)
```

**Timestamps Unix ms ↔ ISO string:**
```ts
// Al escribir: Date.now()
// Al leer (para mantener firmas públicas): new Date(e.updatedAt).toISOString()
```

**Cache tipado explícitamente para evitar error de union type:**
```ts
// CORRECTO — sin esto TS infiere union de todas las tablas
const rondaCache = new Map<string, Doc<'rondas'> | null>()
// INCORRECTO — Map<string, Awaited<ReturnType<typeof ctx.db.get>>>
```

### Patrón de llamada desde Server Actions (Fase 4)

```ts
import { fetchQuery, fetchMutation } from 'convex/nextjs'
import { api } from '@/convex/_generated/api'

// Query
const ronda = await fetchQuery(api.rondas.getRonda, { id: rondaId as Id<'rondas'> })

// Mutation
await fetchMutation(api.fichas.submitFicha, { fichaId: fichaId as Id<'fichasRegistro'> })
```
