# Technical Finding / Milestone

**Context**: Codigo aleatorio automatico por participante, Fases 0 y 1.

## The Discovery

El proyecto ya tenia el campo Convex `rondaParticipantes.participantCode`, por lo que la generacion automatica no requiere cambio de esquema para la Fase 1.

La implementacion base vive en `convex/rondas.ts`:

- `PARTICIPANT_CODE_LENGTH = 6`
- `PARTICIPANT_CODE_ALPHABET = 'ABCDEFGHJKLMNPQRSTUVWXYZ23456789'`
- `PARTICIPANT_CODE_MAX_ATTEMPTS = 20`
- `generateParticipantCode()` genera un codigo aleatorio corto.
- `generateUniqueParticipantCode(ctx, rondaId)` consulta `rondaParticipantes` con el indice `by_ronda`, acumula codigos existentes en un `Set`, reintenta hasta 20 veces y falla con un error claro si no logra un codigo unico.

## Key Files Affected

- `logs/plans/260430_0844_plan_codigo-aleatorio-participantes.md`
