# Technical Finding / Milestone

**Context**: SGC Fase 2 - notificaciones in-app

## The Discovery

El flujo de notificaciones admin ya no depende de pegar manualmente el ID de `rondaParticipantes`. El panel SGC devuelve destinatarios elegibles desde `rondaParticipantes` y la mutacion `crearNotificacion` valida que el participante seleccionado pertenezca a la ronda antes de derivar su correo server-side.

Esto mantiene el respaldo de correo manual para casos excepcionales, pero el flujo normal queda atado a participantes reales de la ronda.
