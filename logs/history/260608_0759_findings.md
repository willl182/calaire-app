# Technical Finding / Milestone

**Context**: SGC production auth integration with WorkOS AuthKit and Convex

## The Discovery

The production SGC flow needed two distinct auth fixes:

1. The Next.js server-side wrapper must obtain the access token through `requireAuth()` and pass that token into `convex/nextjs` calls.
2. The Convex auth config must use the WorkOS JWT issuer that actually signs the token, without over-constraining the second `customJwt` provider with `applicationID`.

After both corrections, the SGC page can authenticate correctly against Convex from production.

## Key Files Affected

- `lib/sgc/index.ts`
- `convex/auth.config.ts`
- `convex/_generated/ai/ai-files.state.json`
