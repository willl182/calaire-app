# Technical Finding / Milestone

**Context**: Correccion de review PR SGC Maestro protv2

## The Discovery

Se resolvio la tanda principal de hallazgos de `cr-rev.md` sin hacer commit ni push. Los cambios fortalecen tres areas:

- Control de flujo Next: `redirect()` no debe ejecutarse dentro de `try/catch` en server actions porque lanza una excepcion de control de flujo.
- Seguridad de datos editables: URLs cargadas por editores deben validarse como `http(s)` antes de persistir y tambien antes de renderizar como links.
- Consistencia Convex: IDs con forma valida no garantizan existencia ni pertenencia; las mutaciones deben validar documento/requisito/version antes de crear relaciones o registros.

## Solution Summary

- Se creo `lib/safe-url.ts` con helpers `normalizeHttpUrl` y `requireHttpUrl`.
- Se agrego filtrado por visibilidad en backend Convex para documentos SGC.
- Se agrego indice version-aware para requisitos normativos y se actualizaron imports.
- Se hizo idempotente el import de mapa por clave natural y se evito borrar relaciones curadas desde el import generico.
- Se valido que descargas de versiones respeten el par `documentoId` + `versionId`.
- Se corrigieron labels accesibles, cobertura agregada, waits de screenshot y detalles menores del prototype/nav.
