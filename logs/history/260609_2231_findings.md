# Technical Finding / Milestone

**Context**: FT2 Expediente SGC Convex production validation

## The Discovery

The previous `FunctionPathNotFound` for `rondas:getRonda` was caused by a stale or mismatched Convex runtime target. The source already exported `getRonda`, and deploying Convex production published the function correctly.

The focused E2E test then failed in production because its hardcoded ronda ID (`kd7b0emdk7cmzp1vn34f2bfv7986bb77`) did not exist in production data. Querying Convex production showed `kd77ck9jqbeafg5g61c7cw0vrh8756qr` as an existing SGC-capable ronda. Updating the test fixture to that ID made the production E2E pass.
