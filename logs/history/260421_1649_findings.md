# Technical Finding — Sistema de diseño "Institutional Gold"

**Context**: Rediseño UI calaire-app — Fases 1–4 completadas

## The Discovery

### Sistema de clases utilitarias en globals.css

Se definieron clases CSS puras (sin Tailwind) como `.card`, `.card-accent`, `.header-bar`, `.btn-primary`, `.btn-outline`, `.numeric` en `app/globals.css`. Esto permite:
- Aplicar el sistema de marca con una sola clase en JSX
- Mantener consistencia sin repetir inline styles ni cadenas largas de Tailwind

### Patrón de migración de headers

El header flotante (`rounded-[2rem] bg-white/90 backdrop-blur`) se reemplazó por `header-bar` (plano, sin sombra, con `border-bottom: 4px solid var(--pt-primary)`). El `header-bar` vive dentro del contenedor de ancho máximo, no abarca full-width de la pantalla.

### Botones pill pequeños vs .btn-outline

Los botones pill de navegación (Participantes / Resultados / Publicar ronda) son demasiado pequeños para el `.btn-outline` estándar. Se mantuvieron como `rounded-full` con clases de hover personalizadas usando CSS vars:
```
hover:border-[var(--pt-primary)] hover:bg-[var(--pt-primary-subtle)]
```

### KPI cards con .card-accent

En la página de resultados, los 3 KPI cards usan `.card-accent` (borde izquierdo dorado 4px). Los números dentro llevan clase `.numeric` (JetBrains Mono, tabular-nums).

### Valores numéricos en tabla de resultados

`valorCell()` fue modificada para retornar `<span className="numeric">` cuando hay valor, permitiendo aplicar JetBrains Mono a x̄, u, y réplicas individuales en la matriz.

### Tailwind CSS 4 — sin config externo

No existe `tailwind.config.ts`. El theme se extiende vía `@theme inline` en `globals.css`. Los CSS vars se consumen en JSX con sintaxis `bg-[var(--nombre)]`.

## Key Files Affected

- `app/globals.css` — sistema de tokens y clases utilitarias completo
- `app/layout.tsx` — fuentes DM Sans + JetBrains Mono
- `app/login/page.tsx` — card-accent, btn-primary
- `app/(protected)/dashboard/page.tsx` — header-bar, btn-primary/outline, card
- `app/(protected)/dashboard/rondas/[id]/participantes/page.tsx` — header-bar, card, table headers dorados
- `app/(protected)/dashboard/rondas/[id]/resultados/page.tsx` — card-accent KPIs, numeric, btn-primary export
