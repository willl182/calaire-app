# Problem Record (Troubleshooting)

**Severity**: Medium

## The Problem

El smoke e2e publico de Fase 2 no pasa con el webServer automatico de Playwright.

Comando reproducido:

```bash
PLAYWRIGHT_START_SERVER=1 pnpm exec playwright test tests/e2e/app.spec.ts --project=chromium
```

Resultado:

- Playwright agota el timeout de 120 segundos esperando el `webServer`.
- Next/Turbopack emite repetidamente:

```text
Error: Can't resolve 'tailwindcss' in '/home/w182/w421'
```

El detalle de resolucion muestra que intenta resolver desde `/home/w182/w421` y rutas del paquete global de Codex, en lugar de resolver limpiamente desde `/home/w182/w421/calaire-app2`.
