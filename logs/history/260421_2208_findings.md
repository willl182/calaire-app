## Corrección realizada

### Error de tipo en `listEnviosPTRound()`

**Problema detectado:**
El tipo `EnvioPTRow` definía las relaciones como objetos individuales, pero Supabase devuelve arrays.

**Solución aplicada:**
```typescript
type EnvioPTRow = {
  // ... otros campos
  ronda_participantes: RondaParticipantePT[]  // Cambiado de objeto a array
  ronda_pt_items: RondaPTItem[]              // Cambiado de objeto a array
  ronda_pt_sample_groups: RondaPTSampleGroup[] // Cambiado de objeto a array
}
```

**Actualización del mapeo:**
```typescript
return rows.map((row) => ({
  pollutant: row.ronda_pt_items[0]?.contaminante.toLowerCase() ?? '',
  run: row.ronda_pt_items[0]?.run_code ?? '',
  level: row.ronda_pt_items[0]?.level_label ?? '',
  participant_id: row.ronda_participantes[0]?.participant_code ?? '',
  replicate: row.ronda_participantes[0]?.replicate_code ?? 0,
  sample_group: row.ronda_pt_sample_groups[0]?.sample_group ?? '',
  mean_value: row.mean_value,
  sd_value: row.sd_value,
}))
```
