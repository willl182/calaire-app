# Technical Finding / Milestone

**Context**: Playwright E2E local para CALAIRE App con WorkOS/AuthKit

## The Discovery

Playwright quedó configurado para dos modos complementarios:

- Smoke público: `pnpm test:e2e` valida que `/login` renderiza correctamente.
- Sesión autenticada local: `pnpm test:e2e:auth:manual` permite que el usuario haga login WorkOS manualmente en Chromium visible y guarda la sesión en `.auth/workos.json`.

Cuando existe `.auth/workos.json`, el proyecto `authenticated-chromium` queda habilitado automáticamente y puede ejecutar pruebas `*.auth.spec.ts` contra rutas protegidas.

También existe un setup con credenciales (`E2E_AUTH_EMAIL` y `E2E_AUTH_PASSWORD`) para entornos locales donde WorkOS no tenga MFA/captcha/bloqueos, pero el flujo recomendado en esta sesión es manual.
