# Technical Finding / Milestone

**Context**: Persistencia del cierre parcial del panel SGC

## The Discovery

Se confirmo un limite estructural del canal actual: `fetchMutation` de `convex/nextjs` solo acepta referencias publicas, asi que no sirve para invocar `internalMutation` de Convex desde server actions. Eso invalida el atajo de seguridad que se intento usar para SGC.

La correccion aplicada en la sesion fue:

- mantener las mutaciones SGC publicas por compatibilidad con el wrapper actual;
- endurecer la action de evidencia con validacion de pertenencia a ronda antes de descargar o retirar;
- documentar la deuda de auth para resolverla con `ConvexProviderWithAuth` + `convex/auth.config.ts` o con un canal server-only diferente.

## Key Files Affected

- `convex/sgc.ts`
- `lib/sgc/index.ts`
- `app/(protected)/dashboard/rondas/[id]/sgc/actions.ts`
- `logs/CURRENT_SESSION.md`
