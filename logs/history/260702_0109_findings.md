# Technical Finding / Milestone

**Context**: Drive documental SGC - Fase 3

## The Discovery

La inicializacion documental puede mantenerse idempotente usando `rondaId + codigo` como llave funcional para raiz, carpetas y documentos esperados. La reparacion debe limitarse a metadatos derivados del catalogo (`parentId`, `nombre`, `fase`, relaciones SGC y `critico`) y no tocar `webUrl`, `definitivo` ni estados operativos para no perder trazabilidad manual.

La criticidad se deriva de `SGC_FORMATOS_FASE_1` solo cuando el recurso tiene `formatoOperativo`; los recursos de evidencia generica quedan no criticos hasta que el catalogo defina una regla explicita.
