# Technical Finding / Milestone

**Context**: Fase 3 SGC - Matriz documental maestra

## The Discovery

Se agrego la primera parte de la matriz documental global de SGC sin romper la interfaz existente del panel:

- `documentosSgc` modela el documento maestro por codigo, proceso, tipo, estado y criticidad.
- `documentoSgcVersiones` conserva el historial de versiones con estado vigente/reemplazada/retirada.
- `convex/sgc.ts` expone queries y mutations para listar la matriz, crear/actualizar documentos y registrar versiones.
- `lib/sgc/index.ts` incorpora helpers tipados para consumir la matriz desde la UI.
- `/dashboard/sgc` agrega un resumen global y formularios de alta/versionado usando los mismos patrones visuales del dashboard existente.

## Key Files Affected

- `convex/schema.ts`
- `convex/sgc.ts`
- `lib/sgc/index.ts`
- `app/(protected)/dashboard/sgc/page.tsx`
- `app/(protected)/dashboard/sgc/actions.ts`
