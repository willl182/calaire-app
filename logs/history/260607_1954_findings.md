# Technical Finding / Milestone

**Context**: SGC documentary close workflow hardening

## The Discovery

The SGC implementation already enforced `activa -> documentacion_pendiente -> cerrada` inside Convex, but the generic dashboard server action still accepted `next_state=cerrada`. Convex would reject that direct transition, but the UI/action copy still represented a legacy close path outside the SGC panel.

## Key Files Affected

- `app/(protected)/dashboard/actions.ts`
- `convex/migrations.ts`
- `app/(protected)/dashboard/rondas/[id]/sgc/page.tsx`

## Result

- `changeRondaStatusAction` now only accepts `activa`; closing must go through the SGC route.
- `insertRonda` migration validator accepts `documentacion_pendiente`.
- Cronograma hito edits preserve the stored `visibleParticipante` checkbox state.
