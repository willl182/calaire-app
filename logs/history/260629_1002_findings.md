# Technical Finding / Milestone

**Context**: SGC de ronda / expediente documental EA-PP-2026-R1

## The Discovery

La vista SGC dentro de una ronda no debia operar como un panel SGC generico ni como un checklist interno calculado. La estructura correcta para el usuario es un expediente documental por secciones basado en `docs/EA-PP-2026-R1`.

El catalogo `SGC_RONDA_ETAPAS` ahora declara las secciones y documentos esperados de la ronda, incluyendo documentos con archivo base disponible, documentos pendientes y formatos que tienen soporte operativo en la app. La pagina de ronda usa ese catalogo para renderizar el checklist real.

## Key Files Affected

- `lib/sgc/catalog.ts`
- `app/(protected)/dashboard/rondas/[id]/sgc/ExpedienteSgc.tsx`
- `app/(protected)/dashboard/rondas/[id]/sgc/page.tsx`
- `app/(protected)/dashboard/rondas/[id]/sgc/actions.ts`
