# Technical Finding / Milestone

**Context**: CodeRabbit fixes on `feat/convex-migration`

## The Discovery

Los findings se pudieron resolver de forma incremental al separar los cambios por riesgo:

1. Primero invariantes Convex y consistencia transaccional.
2. Luego semantica de `null`/`undefined` y schema.
3. Despues formularios y acciones PT.
4. Luego rendimiento y exportacion.
5. Finalmente UI/accesibilidad.

Ese orden permitio mantener `pnpm exec tsc --noEmit`, `pnpm lint` y `pnpm build` verdes tras cada bloque relevante.
