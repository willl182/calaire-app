# Technical Finding / Milestone

**Context**: SGC round plan block catalog refactor and production deployment

## The Discovery

The SGC round plan blocks `a` through `u` had become duplicated between the editor and print view, while the save and print flows also encoded the block order with the magic string `'abcdefghijklmnopqrstu'`. This made the domain model fragile: the app had multiple sources of truth for the same regulatory block list.

The fix was to move the block metadata to `lib/sgc/catalog.ts` as `SGC_PLAN_BLOQUES`, derive `SgcPlanBloqueKey` and `SgcPlanBloques` from that constant, and make the editor, print view, and server action iterate the same catalog.

## Key Files Affected

- `lib/sgc/catalog.ts`

- `app/(protected)/dashboard/rondas/[id]/sgc/plan/print/page.tsx`
