# Technical Finding / Milestone

**Context**: Vista "Cobertura por Ronda" / Dashboard SGC

## The Discovery

The SGC dashboard can now render a new default tab for cobertura cruzada by reusing the existing `listRondasSgcResumen` query. The implementation lives entirely in the frontend:

1. `app/(protected)/dashboard/sgc/SgcTabs.tsx` holds the client-side tab state.
2. `app/(protected)/dashboard/sgc/TableroCoberturaRondas.tsx` renders the reactive matrix with search, hide-closed filtering, sticky columns, status dots, and deep navigation to each round's SGC page.
3. `app/(protected)/dashboard/sgc/page.tsx` keeps the round summary cards above the tab strip and switches the lower section between the new matrix and the existing document matrix.

During validation, `pnpm exec tsc --noEmit` and `pnpm build` both passed after fixing two pre-existing type mismatches in `MatrizInteractiva.tsx` (`notes` -> `notas`, `createdBy` -> `updatedBy`).
