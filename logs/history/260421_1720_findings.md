# Technical Finding: Fase 6 — Logo UNAL + Verificación Visual

**Context**: Fase final del rediseño UI "Institutional Gold" CALAIRE-EA.

## The Discovery

### Logo UNAL
- Fuente: `pt_app/www/logo.png` (7811×3321 RGBA PNG, formato horizontal muy ancho)
- Copiado a `public/logo-unal.png`
- Componente `app/components/LogoUnal.tsx` usa `next/image` con prop `height` y calcula `width` automáticamente (`height * 7811/3321`) para preservar aspect ratio
- Login: `<LogoUnal height={36} />` — encima del título "CALAIRE-EA"
- Dashboard header: `<LogoUnal height={32} />` — encima del texto del header bar

### Verificación visual (revisión de código)
- `globals.css`: paleta correcta, todas las utilidades presentes
- `login/page.tsx`: eliminados inline styles redundantes (`.card-accent` ya define `border-left: 4px solid var(--pt-primary)`)
- `dashboard/page.tsx`: header-bar con borde dorado inferior, logo, títulos con CSS vars
