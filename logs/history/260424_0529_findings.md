# Technical Finding / Milestone: Deploy a producción

**Context**: Deploy inicial de CALAIRE-EA en Vercel

## The Discovery

El deploy en Vercel con WorkOS AuthKit requiere configurar **dos** URLs en el dashboard de WorkOS, no solo el redirect:
1. **Redirect URI**: `https://dominio.vercel.app/auth/callback`
2. **Homepage URL**: `https://dominio.vercel.app` (si se omite, algunos flujos de error redirigen al localhost)

El comando `npx vercel --yes` desde el directorio del proyecto detecta Next.js automáticamente, vincula el repo de GitHub y crea el proyecto en Vercel sin intervención manual.

Las variables de entorno se agregan con `npx vercel env add VAR_NAME production <<< "valor"` sin necesidad de abrir el dashboard de Vercel.

## Key Files Affected

- `.vercel/` — creado por CLI, contiene project ID y org ID
- `logs/CURRENT_SESSION.md` — URLs de producción actualizadas
