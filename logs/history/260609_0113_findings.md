# Technical Finding / Milestone

**Context**: SGC summary production redeploy

## The Discovery

The likely production crash path for `sgc:listRondasSgcResumen` was a brittle assumption in `collectCoverage`: legacy or incomplete `rondaParticipantes` documents could make `p.workosUserId.startsWith('pendiente:')` throw at runtime.

The patch made the coverage calculation defensive around string fields used for participant claim status, participant codes, contaminante labels, and sample group labels. A route-level Next.js error boundary was also added so `/dashboard/sgc` shows a recoverable fallback instead of a black screen if a client-side Convex query throws again.
