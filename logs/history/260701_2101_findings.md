## Metodo util para futuras revisiones

Para validar un "move de dependencias" no basta leer `package.json`: extraer todos los imports npm de `src/` y confirmar que cada uno esta en `dependencies`, y comprobar que `convex/` no requiera el paquete por separado (deploy independiente). Comando base:
`rg -oNI "from ['\"]([a-z@][^'\"]+)['\"]" src -r '$1' | grep -vE '^\.|^@/' | sed -E 's#^(@[^/]+/[^/]+).*#\1#; s#^([^@/][^/]*).*#\1#' | sort -u`
