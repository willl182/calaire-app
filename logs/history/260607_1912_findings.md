# Technical Finding / Milestone

**Context**: Panel SGC por ronda

## The Discovery

Se completo un incremento operativo de Fase 1 para el panel SGC:

- Los hitos ahora pueden actualizarse desde `/dashboard/rondas/[id]/sgc`, incluyendo estado, fechas objetivo/real, responsable, bloqueo de cierre, formato relacionado y notas.
- Las evidencias vigentes ahora pueden descargarse desde una server action con validacion admin previa.
- Las evidencias vigentes ahora pueden retirarse con motivo obligatorio, usando la mutacion Convex existente `retirarEvidenciaVersion`.
- Se corrigio una falla de modelado en `convex/sgc.ts`: `createHitoRonda` y `updateHitoRonda` ya no insertan/parchean campos de transporte (`actor`, `hitoId`) dentro del documento `sgcHitosRonda`.

## Key Files Affected

- `app/(protected)/dashboard/rondas/[id]/sgc/actions.ts`
- `app/(protected)/dashboard/rondas/[id]/sgc/page.tsx`
- `lib/sgc/index.ts`
- `convex/sgc.ts`
