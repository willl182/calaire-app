# Technical Finding / Milestone

**Context**: Migración estructural del repo `calaire-app` a layout T3.

## Milestone

Fases 0 y 1 de `PLAN_MIGRACION_T3.md` completadas y commiteadas en la rama `feature/t3-estructura-segura`.

## The Discovery

- `@t3-oss/env-nextjs` 0.13.x + Zod 4 funcionan con Next.js 16.2.4 usando `runtimeEnv` manual.
- Es necesario excluir `tests/e2e/` de Vitest porque Playwright comparte el directorio `tests/` y Vitest intenta ejecutar los specs de Playwright, fallando con "Playwright Test did not expect test() to be called here".
- Los tests antiguos en `lib/**/*.test.ts` usan `node:test`/`node:assert`; Vitest no los reconoce como suites. Se dejan para Fase 3, cuando se migren a `src/server/<dominio>/`.
- `passWithNoTests: true` evita que `pnpm test` falle mientras no haya tests unitarios en `src/`.
