# Technical Finding: Card PT App + ajustes logo UNAL

**Context**: Dashboard CALAIRE-EA — integración de enlace a herramienta de análisis Shiny.

## The Discovery

### Card PT App (solo admin)
- Ubicada dentro del bloque `{!admin ? ... : (...)}`, **encima** del grid `xl:grid-cols-[1.1fr_1.4fr]`
- Implementada como `<a href="..." target="_blank" rel="noopener noreferrer">` con clase `.card`
- Hover: `hover:border-[var(--pt-primary)]` para feedback dorado consistente con el sistema
- Botón visual: `<span className="btn-primary shrink-0">Abrir →</span>` (no es `<button>` porque el padre ya es el enlace)
- URL: `https://w421.shinyapps.io/pt_app/`

### Ajuste logo login
- Tamaño subido de `height=44` a `height=64` por petición del usuario

### Estructura wrapper necesaria
Al agregar la card antes del grid de rondas, se necesitó un div contenedor extra:
```tsx
{!admin ? (
  <ParticipanteView ... />
) : (
  <div className="grid gap-6">          {/* wrapper nuevo */}
    <a ... />                            {/* card PT App */}
    <div className="grid gap-6 xl:grid-cols-[1.1fr_1.4fr]">  {/* grid rondas */}
      ...
    </div>
  </div>
)}
```
