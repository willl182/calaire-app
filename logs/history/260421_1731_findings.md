# Technical Finding: Logo UNAL — Integración y ajuste final

**Context**: Fase 6 del rediseño UI "Institutional Gold" en CALAIRE-EA (Next.js).

## The Discovery

### Fuente del logo
- `pt_app/www/logo.png` → copiado a `public/logo-unal.png`
- Dimensiones: 7811×3321 px, PNG RGBA, formato horizontal muy ancho

### Componente `LogoUnal`
```tsx
// app/components/LogoUnal.tsx
import Image from 'next/image'

export function LogoUnal({ height = 40 }: { height?: number }) {
  const width = Math.round(height * (7811 / 3321))
  return (
    <Image src="/logo-unal.png" alt="Universidad Nacional de Colombia"
      width={width} height={height} priority />
  )
}
```
- Prop `height` controla el tamaño; `width` se calcula automáticamente para preservar aspect ratio.
- `priority` evita LCP penalty en login (above the fold).

### Posicionamiento
- **Login**: logo **sobre** la card (fuera de ella), centrado, `height=64`. Estructura:
  ```
  div.max-w-sm.flex-col.items-center.gap-8
    LogoUnal height=64
    div.card-accent   ← card de login
  ```
- **Dashboard**: logo dentro del `.header-bar`, encima del bloque de texto, `height=32`.

## Key Files Affected
