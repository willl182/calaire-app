# Technical Finding / Milestone

**Context**: SGC coverage board visual correction

## The Discovery

The production SGC board rendered correctly, but sticky columns and header/control surfaces used translucent backgrounds and hover opacity classes. This created visual overlay artifacts while horizontally scrolling the matrix.

The fix focused on the transparency issue rather than backend/data logic:

- Removed explicit transparency classes from `TableroCoberturaRondas.tsx`.
- Replaced translucent control surfaces with solid `bg-white`.
- Added solid borders to sticky left/right columns.
- Replaced translucent row hover with a solid amber-tinted background.
