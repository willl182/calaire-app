# Technical Finding / Milestone

**Context**: Drive documental SGC - fase 4 manual Google Drive

## The Discovery

El MVP manual no necesita credenciales Google API para operar de forma util: Convex puede guardar URLs de Drive, extraer `driveFileId` o `driveFolderId` desde patrones comunes y auditar cada cambio. Para reemplazos, la opcion implementada es mantener un unico recurso vigente y registrar la trazabilidad completa en `sgcAuditLog`.

Eventos relevantes:

- `sgc.drive.root_registrado`
- `sgc.drive.recurso_enlazado`
- `sgc.drive.definitivo_registrado`
- `sgc.drive.recurso_reemplazado`
