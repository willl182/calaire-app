# Technical Finding / Milestone

**Context**: Migracion T3 Fase 5 / Convex file-based API.

## The Discovery

La Fase 5 quedo implementada moviendo los modulos Convex raiz a carpetas por dominio. En la version actual de Convex, `convex/<dominio>/index.ts` no conserva el path publico `api.<dominio>.*`; se genera como `api.<dominio>.index.*`.

Por eso se actualizaron los consumidores a:

- `api.rondas.index.*`
- `api.sgc.index.*`
- `api.fichas.index.*`
- `api.pt.index.*`
- `api.agent.auth.*` para el antiguo `agentAuth`
