# Technical Finding / Milestone

**Context**: Fase 3 SGC - Matriz documental maestra y validación Playwright

## The Discovery

Se ejecutó con éxito la suite de pruebas e2e de Playwright para SGC (tanto Fase 2 como Fase 3). 
Para evitar errores de timeout y de concurrencia (`ECONNREFUSED` y sobrecarga de CPU en el sandbox), la suite de pruebas de Playwright se ejecutó de forma secuencial (`--workers=1`). Esto garantizó que el servidor local de desarrollo en el puerto 3000 y el backend de Convex en el puerto 3210 pudieran responder a tiempo a todas las peticiones del navegador de Playwright.
