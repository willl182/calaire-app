# Problem Record (Troubleshooting)

**Severity**: Medium

## The Problem

The focused Playwright E2E test could not complete against the local started server because the target Convex deployment returned:

```text
FunctionPathNotFound: Could not find public function for 'rondas:getRonda'.
```

This occurred while rendering `/dashboard/rondas/[id]/sgc`, before the new expediente assertions could run.

## User-Requested Corrections

- Use the Playwright browser configured at `/usr/bin/chromium` for captures or browser verification, not a normal browser.

## The Solution / Workaround

No code workaround was applied because `convex/rondas.ts` already exports `getRonda`. The likely issue is a stale or mismatched Convex deployment/codegen target for the environment used by `.env.local`.

Run Convex sync/deploy for the intended environment, then repeat:

```bash
pnpm test:e2e:start tests/e2e/sgc-fase2.auth.spec.ts
```

## Learnings to Preserve

- If SGC E2E fails before page render with `FunctionPathNotFound`, verify Convex deployment state before debugging the UI.
- The repo Playwright config already points Chromium to `/usr/bin/chromium`; preserve that for future screenshot or E2E work.
