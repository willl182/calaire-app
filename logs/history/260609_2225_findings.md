# Technical Finding / Milestone

**Context**: FT2 Expediente SGC implementation

## The Discovery

The round SGC page can act as the operational expediente without adding new backend APIs. The existing `panel.checklist`, evidence series, evidence version actions, and justification actions were sufficient for the MVP.

The new `ExpedienteSgc` component uses `panel.checklist` as the source of truth, groups the 12 checklist formats by phase, and resolves the relevant existing series or justification by format code. Query parameter `?formato=...` is validated against `SGC_FORMATOS_FASE_1`, then used to expand and highlight the matching expediente item.

Evidence and justification form submissions include `formato_focus`, allowing server action redirects to preserve contextual focus after mutations.
