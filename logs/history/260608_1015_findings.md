# Technical Finding / Milestone

**Context**: WP11 SGC — Plantillas P-20

## The Discovery

El panel SGC tenia enlaces a plantillas P-20 (`/sgc/templates/p20/...`) pero no existian ni los archivos ni la ruta. Se implemento WP11 creando:

1. **5 plantillas Markdown** en `lib/sgc/templates/p20/`:
   - `convocatoria.md`
   - `recordatorio-ficha.md`
   - `recordatorio-envio-resultados.md`
   - `publicacion-resultados.md`
   - `cierre-ronda.md`

2. **Helper de renderizado** en `lib/sgc/templates.ts`:
   - `readTemplate(name)` lee del filesystem en server components.
   - `renderTemplate(template, vars)` reemplaza `{{variable}}` por valores.
   - `buildTemplateVars(ronda, baseUrl)` construye las variables por defecto.
   - Placeholders visibles (ej. `[FECHA LIMITE — COMPLETAR]`) para campos que no estan en el modelo de ronda.

3. **Pagina de visualizacion** en `app/(protected)/dashboard/rondas/[id]/sgc/templates/[name]/page.tsx`:
   - Requiere auth + admin.
   - Renderiza el template con variables de la ronda actual.
   - Muestra vista previa con estilos limpios.
   - Incluye advertencia visible: "En Fase 1 no hay envio automatico de correos."

4. **Boton de copiar** (`CopyButton.tsx`) para portapapeles.

5. **Enlaces corregidos** en el panel SGC para apuntar a `/dashboard/rondas/${id}/sgc/templates/${name}`.

## Key Files Affected

- `lib/sgc/templates/p20/*.md`
- `lib/sgc/templates.ts`
- `app/(protected)/dashboard/rondas/[id]/sgc/templates/[name]/page.tsx`
- `app/(protected)/dashboard/rondas/[id]/sgc/templates/[name]/CopyButton.tsx`
- `app/(protected)/dashboard/rondas/[id]/sgc/page.tsx`
