# Technical Finding / Milestone: Diseño de Vista Cobertura por Ronda

**Context**: SGC / Cobertura por Ronda / Grill-Me Design

## The Discovery

During the grill-me design phase, we discovered that:
1. The query `listRondasSgcResumen` in `convex/sgc.ts` already returns the SGC coverage checklist (`checklist` array containing 12 items with name and status/estado) for each round.
2. The front-end can directly consume this endpoint. No new backend queries or schema modifications are required.
3. We mapped out the UI design: a responsive, horizontally-scrollable matrix (heatmap) displaying rounds as rows and the 12 SGC formats (grouped by phase) as columns, featuring colored semaphore status dots, tooltips, and deep navigation to each round's SGC page.
