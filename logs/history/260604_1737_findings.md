# Technical Finding / Milestone

**Context**: Reutilización de ficha por NIT o correo y despliegue a producción

## The Discovery

Se implementó una búsqueda de plantilla previa en Convex que normaliza el valor de entrada y compara contra `nitLaboratorio` o `correoLaboratorio` en `fichasRegistro`.

Cuando hay coincidencia, el editor admin puede copiar la información completa de la ficha más reciente encontrada, incluyendo:

- datos básicos del laboratorio;
- acompañantes;
- analizadores;
- instrumentos.

La lógica quedó expuesta en la UI admin para que el coordinador pueda reutilizar registros sin reescribir la información.

## Key Files Affected

- `convex/fichas.ts`
- `lib/fichas.ts`
- `app/(protected)/dashboard/rondas/[id]/participantes/[pid]/ficha/FichaAdminEditor.tsx`
