# Technical Finding / Milestone

**Context**: Despliegue productivo de SGC Fase 1 con Convex AuthKit.

## The Discovery

El despliegue de Convex con `convex/auth.config.ts` no basta con recibir `WORKOS_CLIENT_ID` desde el proceso local. Convex valida que las variables usadas por el auth config existan en el deployment hospedado.

El primer `convex deploy` fallo con:

```txt
Environment variable WORKOS_CLIENT_ID is used in auth config file but its value was not set.
```

La solucion fue configurar la variable en el deployment productivo:

```bash
pnpm exec convex env set --deployment steady-kiwi-725 WORKOS_CLIENT_ID "$WORKOS_CLIENT_ID"
```

Luego el deploy de Convex completo correctamente y aplico los indices nuevos de `sgcJustificaciones`.
