# Technical Finding / Milestone

**Context**: Validacion post-login en produccion

## The Discovery

El usuario confirmo que pudo iniciar sesion correctamente en `https://calaire-app.vercel.app` despues de corregir las variables de WorkOS/Vercel.

Esto valida que:

- `NEXT_PUBLIC_APP_URL` esta apuntando al dominio canonico de produccion.
- `NEXT_PUBLIC_WORKOS_REDIRECT_URI` esta apuntando a `/auth/callback` en produccion.
- El redeploy `dpl_6xA1V1QyfF5L9kKkUUsok6NeHXqn` resolvio el error post-login.
