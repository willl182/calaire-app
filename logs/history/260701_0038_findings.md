# Technical Finding / Milestone

**Context**: Fase 3 - seguridad Convex

## The Discovery

`convex-test` puede ejecutarse dentro de la suite Vitest del repo con `withIdentity()` para simular usuarios autenticados o, sin identidad, para validar que funciones protegidas rechazan como se espera. Para que el loader de módulos no se autoimporte, conviene sacar `import.meta.glob('./**/*.ts')` a un map y eliminar el archivo de test del glob.

## Key Files Affected

- `convex/access.test.ts`
- `convex/vite-env.d.ts`
- `vitest.config.ts`
